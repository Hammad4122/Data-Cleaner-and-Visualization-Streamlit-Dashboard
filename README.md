# Data-Cleaner-and-Visualization-Streamlit-Dashboard

A **Streamlit-based interactive dashboard** for **uploading, cleaning, and visualizing datasets** (CSV & Excel) with **minimal coding effort**. Quickly prepare your data and generate insightful visualizations.

---

## Features

### Data Upload & Conversion
- Upload **CSV**, **XLSX**, or **XLS** files.  
- Automatic detection of file type.  
- Convert uploaded files into a **pandas DataFrame** for easy processing.

### Data Cleaning
- **Detect and remove duplicate rows** automatically.  
- **Handle missing values** using **Forward Fill (ffill)** or **Backward Fill (bfill)**.  
- Download the cleaned dataset as a **CSV**.

### Data Visualization
- Visualize data with charts: **Bar**, **Line**, **Scatter Plot**.  
- Explore **individual columns** or perform **grouped aggregations**: `count`, `sum`, `mean`, `max`, `min`.  
- Customizable **X-axis** and **Y-axis** selections.

### User-Friendly Interface
- **Sidebar control panel** for switching between **Visualization** and **Cleaning** modules.  
- Interactive guidance for **column selection**, **grouping**, and **aggregation operations**.
